# Dashboard Financiero v8.1 — Lógica aclarada y archivos limpios

**Conceptos clave**
- **Ingreso asignado (Bruto por Owner)**: suma de % por roles (15/18/50/3/4/10) que correspondan al Owner seleccionado (o total si no se filtra Owner).
- **Previsión (Retención 14,5%)** aplicada sobre el Ingreso asignado.
- **Ingreso líquido** = asignado − 14,5%.
- **Deuda Financiera**: base $31.000.000; cargo mensual $6.000.000; cada mes se descuenta el **Ingreso líquido del Equipo H&A** (85,5% del asignado en facturas Pagadas).

**Incluye**
- Filtros dinámicos (Año facturación, Año pago, Estatus, Owner con “Equipo H&A”).
- KPIs y Planning (gráficos comprimidos, height=35).
- Listados por estatus en disposición vertical.
